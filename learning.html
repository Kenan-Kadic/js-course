How to learn Coding

Big Mistakes

  • You have no plan at all
    • You need a real plan with measurable and achievable goals
    Ask yourself why am I even doing this at all?
    What new project would you like to build?
  • Copy and paste code without knowing how it works
    • You need to actually type out the code not copy paste
    You need to understand the code you use
  • Never doing any challenges to try the code you learned
    • You need some challenges to actually try the stuff you learn
    www.codewars.com has challenges
  • Never doing any practice
  • Never comes up with his own project ideas
  • Become frustrated easily
  • Never learn only by yourself
  • Thinks you can never learn it all so there is no point in trying
    • You will never know everything about coding and that is normal
  • HE NEVER SHARED THE CODE HE LEARNED WITH OTHERS TO GET FEEDBACK
  • He finished a few courses but can't build a single thing on his own
